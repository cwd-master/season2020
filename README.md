## 2020년도 웹디자인기능사 공개문제 자료
``` 웹디자인기능사(Craftman Web Design) 공개문제 소스 및 풀이 자료```
## 최신 업데이트(2020.06.28)

1. 동등한 소프트웨어 허용 범우 : Visual Studio Code 등
2. 시험장 기본시설 이외의 동등한 소프트웨어, 폰트 등 (정품에 한함)을 사용하고자 할 경우 수험자가 지참하여 시험 시작 전 감독위원의 입회하에 설치 할 수 있습니다.
3. 관련 동등한 소프트웨어 설치 범위 (예: Visual Studio Code )를 설치 할 수는 있으나, 관련된 플러그 인(별도 응용, 추가 기능)은 설치 할 수 없습니다.
4. 시험장은 인터넷이 차단되어, 웹사이트 접속 및 클라우드 서버 접속 방식의 프로그램은 설치 할 수 없습니다.  (반드시 정품 라이센스 지참)
5. 그 외 무료폰트, 프리웨어 소프트웨어(플러그 인 포함)는 설치 할 수 없습니다.
6. 수험자가 지참한 마우스 등 입력장치는 사용할 수 없습니다.
7. 지참준비물 이외의 참고 자료 및 저장매체 등 어떠한 물품이라도 시험 중 지참 할 수 없으며, 위반 시는  실격, 활용 시에는 부정행위자로 처리합니다.
